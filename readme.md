# EventHub
## Getting Started
* run ```npm install ``` in public folder
* create venv and install django in server folder
* change the value according to your database
 ```
 DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'eventhub',
        'USER': 'root',
        'PASSWORD': 'password',
        'HOST': 'localhost',
        'PORT': '3306',
    }
}
```
